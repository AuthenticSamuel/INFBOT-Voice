  <h2>Welcome to INFBOT's GitHub page!</h2>
  <div>
    <p><b>INFBOT Voice Channels</b>, most commonly known as <b>INFBOT</b>, is a Discord bot that helps you create <b>automatic voice channels</b> to avoid cluttered channel lists in your servers.</p>
    <p>All commands are preceded by either the <em>GLOBAL</em> prefix: <b>infbot/</b> or your <em>LOCAL</em> prefix: <b>inf/</b> (default).</p>
  </div>
  <div>
    <h3>Currently available commands are:</h3>
    <ul>
      <li>- [x] <b>help</b> ‚û° Sends you the help menu</li>
      <li>- [x] <b>bot</b> ‚û° Gives you information about INFBOT and the latest patchnotes</li>
      <li>- [x] <b>user</b> ‚û° Gives you information about you or the user you mentionned</li>
      <li>- [x] <b>server</b> ‚û° Gives you information about the server you are on</li>
      <li>- [x] <b>prefix</b> ‚û° View the current <em>GLOBAL</em> & <em>LOCAL</em> prefixes</li>
      <li>- [x] <b>setprefix</b> ‚û° [Owner] Change your server's <em>LOCAL</em> prefix</li>
    </ul>
    <h3>INFBOT Voice Channels (INFVCs):</h3>
    <ul>
      <li>- [x] <b>status</b> ‚û° Shows you if INFBOT's Automatic Voice Channels are active or not</li>
      <li>- [x] <b>setup</b> ‚û° [Owner] Sets the bot up for automatic voice channels</li>
      <li>- [x] <b>unsetup</b> ‚û° [Owner] Gets rid of INFBOT's automatic voice channels</li>
      <li>- [x] <b>channelinfo</b> ‚û° View information on the voice channel you're in</li>
      <li>- [x] <b>setbitrate</b> ‚û° Set your INFVC's bitrate</li>
      <li>- [x] <b>setuserlimit</b> ‚û° Set your INFVC's user limit</li>
      <li>- [x] <b>lock</b> ‚û° Makes your INFVC private, disallowing new users to join (Admins can bypass this restriction)</li>
      <li>- [x] <b>unlock</b> ‚û° Makes your INFVC public, allowing new users to join</li>
    </ul>
  </div>
